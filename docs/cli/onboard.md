# `clawdbot onboard`

Interactive onboarding wizard (local or remote Gateway setup).

Related:
- Wizard guide: [Onboarding](/start/onboarding)

## Examples

```bash
clawdbot onboard
clawdbot onboard --flow quickstart
clawdbot onboard --mode remote --remote-url ws://gateway-host:18789
```
